AE 无法启动 142 报错，如何解决？

  • 发布 2026-07-14
  • 阅读

最近，电脑上的 AE 突然无法启动了。提示应该程序错误(0xc0000142)

我尝试卸载，重装，还是一样的报错，奇了怪了，明明前一天还能正常使用的。

根本原因：系统 VC++ 运行库 DLL 文件损坏/被替换

报错弹窗：

先卸载电脑里的 Microsoft Visual C++ 2022，再下载新的安装。（不卸载直接安装，会得到这个安装失败）

下载最新的 Microsoft Visual C++ Redistributable：

  • x64: https://aka.ms/vs/17/release/vc_redist.x64.exe
  • x86: https://aka.ms/vs/17/release/vc_redist.x86.exe

安装好就可以启动了。好像两个都要安装。虽然只安装一个 x64 AE 可以启动，但是后面电脑重启，其它软件一直报错，把 x86 的也安装就正常了。
